# Code of Conduct

This project is committed to a respectful and welcoming community for everyone.

## Expected Behavior

- Be respectful and constructive in discussions.
- Assume good intent and ask for clarification before escalating.
- Give actionable technical feedback.
- Help newcomers understand context and project conventions.

## Unacceptable Behavior

- Harassment, hate speech, threats, or discrimination.
- Personal attacks, trolling, or repeated hostile behavior.
- Sharing private information without consent.
- Disruptive behavior that blocks healthy collaboration.

## Enforcement

Project maintainers are responsible for interpreting and enforcing this Code of Conduct.

Reports are reviewed case by case. Maintainers may take any action they consider appropriate, including warning, temporary restriction, or permanent ban from participation.

## Reporting

If you experience or witness behavior that violates this Code of Conduct, contact the repository owner privately through GitHub: https://github.com/ignaciogarcia-dev
